# Размещение на Railway.app

## Шаги:
1. Создайте аккаунт на railway.app
2. Новый проект → Deploy from GitHub repo
3. Выберите ваш репозиторий
4. Добавьте переменную окружения:
   - Name: TELEGRAM_TOKEN
   - Value: ваш_токен_бота
5. Railway сам всё установит и запустит

## Преимущества:
- Полностью бесплатно
- Автоматический деплой
- Не нужно следить
